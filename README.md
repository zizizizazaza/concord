# Concord — Sovereign Policy Simulation

A landing-page prototype for **Concord**, a sovereign policy-simulation concept for governments:
test a policy in simulation before it is enacted, let a council of AI agents deliberate, and
produce a verifiable, human-reviewed record.

`index.html` is the page (open it directly, or serve the folder with any static server).

> Illustrative product preview. All figures are synthetic and not connected to live government data.
